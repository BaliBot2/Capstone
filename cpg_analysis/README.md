# CPG Analysis Scripts

This directory contains scripts and results related to the hierarchical Leiden community analysis of the CPG.

## Files
-   `hierarchical_leiden.py`: Main script to perform recursive Leiden community detection.
-   `leiden_hierarchy.json`: The output JSON containing the community structure (Layers 0-4).
-   `inspect_results.py`: Script to print a summary of the hierarchy.
-   `analyze_subsystems.py`: Script to analyze the top-level communities (Layer 1).
-   `analyze_cpg_nodes.py`: Script to inspect node properties in the raw CPG.
